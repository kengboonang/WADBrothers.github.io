Font Awesome Icons

To add icons using Font Awesome, go to https://fontawesome.com/icons and search for the icon you want

Remember to click the Free button (paid ones don't work)

Copy the HMTL code snippet: (eg. '<i class="fa-solid fa-circle-user"></i>')
